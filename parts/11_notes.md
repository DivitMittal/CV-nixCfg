## Notes for CV Generation

### Tailoring Recommendations

**For ML Engineer / Research Roles:**
- Lead with: Professional Summary emphasizing AI/ML + Research Experience
- Highlight: MFIF research, HybridTransformer, Computer Vision projects
- Emphasize: PyTorch, Transformers, ViT, HuggingFace ecosystem

**For DevOps / SRE / Infrastructure Roles:**
- Lead with: Professional Summary emphasizing Infrastructure + Tweag consulting
- Highlight: OS-nixCfg, Tweag consulting projects, UbiOps experience
- Emphasize: Nix/NixOS, CI/CD, Kubernetes, reproducible builds

**For Full-Stack / Software Engineer Roles:**
- Lead with: Professional Summary emphasizing versatility
- Highlight: Diverse project portfolio, full-stack capabilities
- Emphasize: Python, TypeScript, Web development, API design

**For Functional Programming / Systems Roles:**
- Lead with: Professional Summary emphasizing FP expertise
- Highlight: hs-faust, OS-nixCfg, type-safe system design
- Emphasize: Haskell, Nix, FFI, DSL design, type systems

### Impact Metrics to Add (When Available)
- **OS-nixCfg:** Number of configurations, build time, reproducibility metrics
- **MFIF:** Model accuracy, inference time, dataset size, download counts
- **hs-faust:** LOC, API coverage, performance benchmarks
- **Tweag Consulting:** Specific % improvements in build times, cost savings
- **BobbleAI Datathon:** Processing time, data volume handled, cost optimization

### Sections to Customize Based on Role
1. Reorder skills to match job requirements
2. Expand relevant projects, condense others
3. Add specific metrics relevant to the role
4. Adjust professional summary to match role focus
5. Consider creating separate CVs for different career tracks
