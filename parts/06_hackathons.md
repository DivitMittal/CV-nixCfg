## Hackathon Awards & Competitions

### BobbleAI Datathon Winner
**BobbleAI Keyboard** | Gurgaon, Haryana, India
*September 2024*

- Processed raw product data from BobbleAI Keyboard (70M+ users) - 8 million event records from 5 days before July 1, 2024
- Extracted & transformed JSON, preprocessed with ML-filtering and stored as structured Apache Parquet on AWS S3
- Utilized serverless AWS Lambda functions & PySpark on AWS Glue for scalable ETL processing
- **Impact:** Enabled data-driven churn analysis for improving user retention
- **Tech Stack:** AWS Lambda, AWS Glue, PySpark, AWS S3, Apache Parquet, Python

### Proglint's Computer Vision 2k23 National Hackathon Winner
**Alliance University** | Anekal, Karnataka, India
*October 2023*

**CARLA Autonomous Driving System**
- Leveraged high-fidelity CARLA simulator to create diverse driving scenarios for autonomous vehicle systems with RGBA & LiDAR sensors
- Utilized deep learning techniques (CNNs) for semantic segmentation to improve vehicle's understanding of surroundings and maneuvering
- **Link:** https://github.com/DivitMittal/CARLA-Autonomous-Driving

**Driver Drowsiness Detection System**
- Built system using Siamese Neural Networks and computer vision techniques (Haar cascade wave filters) to monitor driver's facial features in real-time
- Analyzes eye states & mouth movements to identify potential drowsiness and enhance road safety
- **Impact:** Real-time fatigue detection for preventing road accidents
- **Link:** https://github.com/DivitMittal/Driver-Drowsiness-Detection
